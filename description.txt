Початкове налаштування для системи (разово):
 1. Скачати та встановити node.js (рекомендовану LTS)
 2. Встановити gulp глобально [npm i gulp-cli -g]
    i = install; -g = --global

Налаштування локально для проекта
 1. Ініціалізувати проект (створити package.json) [npm init], 
    вказати стартові параметри або змінити їх в створеному файлі:
    "main": "gulpfile.js" - стартовий файл проекту
    "type": "module" - дозволяє імпортувати новіші версії
 2. Встановити gulp локально для проекта [npm i gulp -D]
    -D - зберегти значення в "package.json"/"devDependencies:{}"

Створення файлової структури:
 1. Створення основного файлу "gulpfile.js"
 2. Створення вихідної папки (src)
 3. Створення папки для роботи з gulp (gulp)
   3.1 Папка налаштувань (config)
     - ftp.js - налаштування вивантаження результату на ftp
     - path.js - шляхи файлів проекта
     - plugins.js - інформація про загальні плагіни
   3.2 Папка завдань (tasks)

Запуск файлу сценарія: [gulp]. 
Якщо задача не зупиняється автоматично, ctrl+C - зупинити

Для нового проєкта:
- скопіювати папку gulp;
- створити відповідну структуру src;
- скопіювати gulpfile.js та package.json
- інсталювати пакети [npm i] (за package.json інсталює все)

Зборка включає в себе копіювання зі спостерігачем, обробку html, scss, js, локальний сервер з оновленням сторінки
